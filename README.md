## Requirements
- NumPy
- NLTK
- PyThaiNLP (2.3.1 Recommended)

## Installation
    git clone https://github.com/TeaKatz/NLP_Preprocessors
    cd NLP_Preprocessors
    pip install --editable .

## Uninstallation
    pip uninstall nlp-preprocessors
